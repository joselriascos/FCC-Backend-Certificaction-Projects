URL Shortener Microservice

Short URL Creation

      
        POST [base url]/api/shorturl
        Content-Type: application/json

        {
        "url": "https://joselriascos-portfolio.netlify.app"
        }

Example Usage:

[base url]/api/shorturl/2

Will Redirect To:

https://joselriascos-portfolio.netlify.app/
